📁 product (Root Object) — [The main object containing all product details]
│
├── 📄 id ─────────────────── [Unique identification ID in Shopify GraphQL format]
├── 📄 title ──────────────── [The product name shown to customers on the website]
├── 📄 handle ─────────────── [The URL slug used in the website link for this product]
├── 📄 vendor ─────────────── [The brand, manufacturer, or seller name]
├── 📄 productType ────────── [The broad product category, e.g., "Functional Food"]
├── 📄 availableForSale ───── [Overall stock status: true = in stock, false = out of stock]
├── 📄 isGiftCard ─────────── [Identifies if it's a digital gift card: true or false]
├── 📄 createdAt ──────────── [Timestamp when the product was first created in the system]
├── 📄 updatedAt ──────────── [Timestamp when any product detail was last updated]
├── 📄 publishedAt ────────── [Timestamp when the product was made live on the website]
├── 📄 onlineStoreUrl ─────── [The direct web link to buy this product]
├── 📄 description ────────── [Plain text description summarizing product benefits]
├── 📄 descriptionHtml ────── [The description written in HTML format for layout style]
│
├── 📂 tags [] ────────────── [List of keywords used for marketing or website filtering]
│    └── 📄 0 ─────────────── ["newi2i" - The first tag attached to the product]
│
├── 📂 category ───────────── [The standardized category according to Shopify Taxonomy]
│    ├── 📄 id ────────────── [The official category ID code, e.g., fb-1-11]
│    └── 📄 name ──────────── [The official category name, e.g., Powdered Beverage Mixes]
│
├── 📂 seo ────────────────── [Search Engine Optimization settings for Google Search]
│    ├── 📄 title ─────────── [Custom title for Google search results; null = uses product title]
│    └── 📄 description ───── [Custom summary under Google link; null = uses default description]
│
├── 📂 featuredImage ──────── [The main display/cover photo of the product]
│    ├── 📄 id ────────────── [Unique image identification ID]
│    ├── 📄 url ───────────── [The direct internet URL link where the image file is hosted]
│    ├── 📄 altText ───────── [Text description of image for slow internet or screen-readers; null = empty]
│    ├── 📄 width ─────────── [The width of the image measured in pixels]
│    └── 📄 height ────────── [The height of the image measured in pixels]
│
├── 📂 images ─────────────── [The entire photo gallery/slideshow for the product page]
│    └── 📂 nodes [] ──────── [An array/list containing all additional images]
│         └── 📂 [0, 1, 2...] [Individual photo blocks with id, url, altText, width, and height fields]
│
├── 📂 priceRange ─────────── [The current selling price range of the product]
│    ├── 📂 minVariantPrice ─ [The lowest variant price with amount and currencyCode]
│    └── 📂 maxVariantPrice ─ [The highest variant price with amount and currencyCode]
│
├── 📂 compareAtPriceRange ── [The original, crossed-out retail price (MRP) range]
│    ├── 📂 minVariantPrice ─ [Old minimum price; 0.0 means there is currently no discount active]
│    └── 📂 maxVariantPrice ─ [Old maximum price; 0.0 means there is currently no discount active]
│
├── 📂 options [] ─────────── [The choice groups available to customers like Size or Color]
│    └── 📂 node ──────────── [The first option group block]
│         ├── 📄 id ───────── [Unique identification ID for this specific option group]
│         ├── 📄 name ─────── [The name of the option, e.g., "Title", "Size", or "Flavor"]
│         └── 📂 optionValues [] [The actual values inside it, e.g., "Default Title", "Small", "Medium"]
│
├── 📂 collections ────────── [Lists all store categories/collections this product belongs to]
│    └── 📂 nodes [] ──────── [An array/list of all assigned collections]
│         └── 📂 node ─────── [A single collection group data block]
│              ├── 📄 id ────────────── [Unique ID of this collection group in the system]
│              ├── 📄 title ─────────── [The display name of the collection, e.g., "Trending Now"]
│              ├── 📄 handle ────────── [The URL slug for the specific collection page]
│              ├── 📄 description ───── [Intro text for the collection page; empty string "" if blank]
│              └── 📄 onlineStoreUrl ── [The direct web link to view the entire collection page]
│
└── 📂 variants ───────────── [The actual physical items tracked in inventory for fulfillment]
     └── 📂 nodes [] ──────── [A list containing all the specific variants available]
          └── 📂 node ─────── [The technical data block of a single product variant]
               ├── 📄 id ────────────────── [Unique variant ID used by backend to process the checkout]
               ├── 📄 title ─────────────── [Variant name; e.g., "Default Title" if no other choices exist]
               ├── 📄 sku ───────────────── [Stock Keeping Unit: Owner's internal warehouse code]
               ├── 📄 barcode ───────────── [Universal barcode number (UPC/EAN) used for physical scanning]
               ├── 📄 availableForSale ──── [Confirms if this exact variant is available for purchase right now]
               ├── 📄 currentlyNotInStock ── [Checks if item is out; false means it IS currently in stock]
               ├── 📄 requiresShipping ──── [Confirms if physical shipping is required; true/false]
               ├── 📄 taxable ───────────── [Confirms if sales tax or GST should be added during checkout]
               ├── 📄 weight ────────────── [The numerical weight value of the item, e.g., 0.07]
               ├── 📄 weightUnit ────────── [The unit used for weight measurement, e.g., KILOGRAMS]
               ├── 📂 price ─────────────── [The specific fixed price and currency code for this variant]
               ├── 📄 compareAtPrice ────── [The original crossed-out price for this variant; null = no discount]
               ├── 📂 image ─────────────── [The specific image that loads when this variant is selected]
               └── 📂 selectedOptions [] ── [Maps this variant back to its matching option values]
